1. 프로젝트 소개

이 프로젝트는 웹사이트를 깃허브를 통해 배포하는 과정을 담은 것이다.

2. 개발 목표

간단한 구조를 기반으로, 향후 확장 가능한 보고서형 웹사이트를 구현한다.

3. 결과물

HTML, CSS로 구성된 정적인 웹페이지로 GitHub Pages를 통해 배포되었다.

4. 결론 및 느낀점

배포 과정이 생각보다 단순했으며, 디자인을 입히는 재미가 있었다.
